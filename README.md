# 🚢 Titanic Survival Analysis (EDA with Python & Seaborn)

## 📌 Project Overview
This project explores the famous Titanic dataset using Python data analysis libraries.  
The goal is to understand which factors influenced passenger survival and present insights through visual storytelling.

---

## 🎯 Objective
To analyze survival patterns on the Titanic and identify how variables such as gender, passenger class, and age affected survival outcomes.

---

## 📂 Dataset
The dataset used in this project is the built-in **Seaborn Titanic dataset**, which includes information about passengers such as:

- Age
- Sex
- Passenger class (1st, 2nd, 3rd)
- Fare
- Survival status
- Family aboard

---

## 🛠️ Tools & Libraries Used
- Python 🐍
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## 📊 Exploratory Data Analysis (EDA)

The analysis includes the following visualizations:

### 1. Overall Survival Distribution
- Count plot showing number of survivors vs non-survivors

### 2. Survival by Gender
- Comparison of survival rates between male and female passengers

### 3. Survival by Passenger Class
- Analysis of socio-economic impact on survival

### 4. Age Distribution and Survival
- Relationship between passenger age and survival outcomes

### 5. Multi-Factor Analysis
- Interaction between sex, class, and age using Seaborn catplot

---

## 🔍 Key Insights

- Overall survival rate was approximately **38%**
- Females had a significantly higher survival rate than males
- First-class passengers had a higher chance of survival
- Younger passengers showed relatively better survival outcomes
- Survival depended on a combination of factors, not a single variable

---

## 📌 Conclusion
The analysis shows that survival on the Titanic was strongly influenced by gender, passenger class, and age.  
Socio-economic status and demographic factors played a major role in determining survival outcomes.
